# colony-voting-mp4s

Short-lived preview mp4s for the colony-ai voting site. Files rotate as voting windows open/close.

Public by design — content here becomes YouTube-public within 1-3 days. No VPS IP leakage; no secret material.

Writers: heartbeat (`push-to-voting.py`) + architect.
Readers: voting site (raw.githubusercontent.com URLs), registered voters.
